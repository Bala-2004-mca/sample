# Pylearn - Python Quiz App

Pylearn is a Flutter-based mobile application designed to help users learn and test their knowledge of Python programming. The app features a series of quizzes covering various topics in Python, from basic syntax to more advanced concepts.

## Features

*   **Interactive Quizzes:** Engage with a variety of multiple-choice questions.
*   **Unit-Based Learning:** Quizzes are structured into units, allowing for progressive learning.
*   **Immediate Feedback:** Get instant results after completing each quiz.
*   **Clean and Intuitive UI:** A user-friendly interface for a smooth learning experience.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Flutter SDK: Make sure you have the Flutter SDK installed. For more information, see the [Flutter documentation](https://flutter.dev/docs/get-started/install).

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username_/pylearn.git
    ```
2.  Install packages
    ```sh
    flutter pub get
    ```
3.  Run the app
    ```sh
    flutter run
    ```
